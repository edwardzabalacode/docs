## Configuración del Sistema

El módulo de parámetros te permite configurar las diferentes opciones del sistema para su funcionamiento.

### Configuración de Telegram

1. **Configuración Básica**
   - Bot Token: Token de acceso al bot de Telegram
   - Chat ID: Identificador del chat donde se enviarán los mensajes
   - Usar Bot de Telegram: Activar/desactivar el envío de mensajes

2. **Formato de Mensajes**
   - Selecciona el formato de mensaje (Básico, Detallado o Separado)
   - Mostrar Gastos: Opción para incluir gastos en los mensajes

3. **Formatos Disponibles**

   ```
   Total: 328.22
   ```

### Configuración de Proyectos

#### Configuración Dating
- **Fee Curator**: Porcentaje de comisión para el curador
- **Fee Points**: Porcentaje de comisión en puntos
- **Fixed Expense**: Gasto fijo del proyecto

#### Configuración Prime
- **Fee Curator**: Porcentaje de comisión para el curador
- **Fee Points**: Porcentaje de comisión en puntos
- **Fixed Expense**: Gasto fijo del proyecto

### Proceso de Guardado

1. Realiza los cambios necesarios en los campos
2. Verifica que los valores sean correctos
3. Haz clic en "Guardar Configuración"
4. Espera la confirmación de guardado exitoso
